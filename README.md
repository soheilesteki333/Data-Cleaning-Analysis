# Retail Sales Data Cleaning & Analysis

This project demonstrates a practical data cleaning and exploratory analysis workflow using Python and pandas on a retail sales dataset.

## Key Tasks
- Data inspection
- Missing value analysis
- Business-rule validation
- Data cleaning
- Feature engineering
- Monthly sales analysis
- Category performance analysis

## Tools Used
- Python
- Pandas
- Jupyter Notebook

## Key Insights
- January generated the highest revenue
- Butchers was the top-performing category
- Missing transactional values were reconstructed using business logic
